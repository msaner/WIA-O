about scrollup

  • Developed by Mark Goodyear - markgoodyear.com
  • Lightweight jQuery plugin that works on all websites.
  • Fully customizable and comes with multiple theme style sheets.

Installing scrollup

Download it from Mark Goodyears website here.

The webpage will have all the information you need to get you started. Including a demo, link to the GIT repo and explanations of the plug ins various features.

the jQuery



$(function(){
  $.scrollUp();
});



$(function () {
  $.scrollUp({
    scrollName: 'scrollUp', // Element ID
    topDistance: '300', // Distance from top before showing element (px)
    topSpeed: 300, // Speed back to top (ms)
    animation: 'fade', // Fade, slide, none
    animationInSpeed: 200, // Animation in speed (ms)
    animationOutSpeed: 200, // Animation out speed (ms)
    scrollText: 'Scroll to top', // Text for element
    activeOverlay: false, // Set CSS color to display scrollUp active point, e.g '#00FFFF'
  });
});
